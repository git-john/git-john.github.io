mac,xcode快捷键大全

  • 2019-02-19

mac文本操作

  • delete 向后删除
  • Fn+delete 向前删除
  • 撤销： Command-z
  • 回退：shift+command+z
  • 剪切：command + x
  • copy：command + c
  • 粘贴：command + v
  • 全选：command + a
  • 保存：command + s
  • 查找：command + f
  • 截取选中区域并保存到文件：command + shift + 4
  • 截取全部屏幕并保存到文件：command + shift + 3
  • 截取全部屏幕到剪贴板： command + shift + control + 3
  • 打开强制退出窗口：command + option + esc
  • fn-上箭头 向上滚动一页（Page Up）
  • fn-下箭头 向下滚动一页（Page Down）　　
  • fn-左箭头 滚动至文稿开头（Home）　　
  • fn-右箭头 滚动至文稿末尾（End）　　
  • Command-右箭头 将光标移至当前行的行尾　　
  • Command-左箭头 将光标移至当前行的行首　　
  • Command-下箭头 将光标移至文稿末尾　　
  • Command-上箭头 将光标移至文稿开头　　
  • Option-右箭头 将光标移至下一个单词的末尾　　
  • Option-左箭头 将光标移至上一个单词的开头　　
  • Control-A 移至行或段落的开头

在Finder中：

  • Command-Option-V 剪切文件　　
  • Command-Shift-N 新建文件夹（New）　　　
  • Command-上箭头 打开包含当前文件夹的文件夹，相当于Windows里的“向上”　　
  • Command-Delete 将文件移至废纸篓　　
  • Command-Shift-Delete 清倒废纸篓　　
  • 空格键 快速查看选中的文件，也就是预览功能

  • 运行与停止

    运行：command + r. 停止： command

  • 调试
